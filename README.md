# Log Reader
a qt based log reader to read spd log more efficiently

## Dependencies
- Qt6.2
- CMake (3.22)
- spdlog (1.9.2)
- boost (1.77.0)

## Features
- Open log file
- Filter log
- Highlight log
- Write user action to log and status bar
- Save user filter and highlight settings
